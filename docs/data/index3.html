Digital Twins for WiFi 7 Security

Security testing and threat prediction for IEEE 802.11be Multi-Link Operation using ns-3 and Graph Neural Networks

Project Overview

This project introduces a high-fidelity Digital Twin (DT) framework for WiFi 7 (IEEE 802.11be) networks supporting Multi-Link Operation (MLO). The DT enables scalable security testing and serves as a data generation engine for machine learning–based threat prediction.

Methodology

Digital Twin & Simulation

A custom ns-3 simulation models WiFi 7 MLO behavior and backoff manipulation attacks. KPIs are collected across PHY, MAC, and network layers in time-windowed JSON format.

Machine Learning

Graph Neural Networks implemented using PyTorch Geometric classify network states as Normal or Attack scenarios, enabling early threat prediction.

Results & Findings

  • Clear degradation in throughput and delay under backoff manipulation attacks
  • Distinct KPI patterns between normal and malicious MLO behavior
  • High classification accuracy achieved using GNN-based models
